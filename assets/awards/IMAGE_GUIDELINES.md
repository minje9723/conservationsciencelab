# 수상 상장 이미지 가이드라인

## 📐 최적 이미지 해상도

### 권장 해상도
- **최소**: 600 x 850px (폭 x 높이)
- **권장**: 800 x 1130px
- **최적**: 1000 x 1414px

### 이미지 비율
- **A4 비율** (1:1.414 - 세로가 더 긴 형태)
- 대부분의 상장이 A4 크기이므로 이 비율 권장
- 예시: 800px(폭) x 1130px(높이)

## 🎯 해상도 선택 이유

### 1. 상장 형태 반영
- 대부분의 상장은 A4 용지 비율 (210mm x 297mm)
- 세로로 긴 형태가 자연스러움

### 2. 카드 레이아웃 최적화
- 데스크톱: 350px 폭 컨테이너에 표시
- 이미지가 너무 크면 로딩 시간 증가
- 적절한 해상도로 선명함과 성능 균형

### 3. 가독성 확보
- 상장의 텍스트가 선명하게 보여야 함
- 800px 이상 권장 (확대 시에도 읽기 가능)

## 📁 파일 형식 및 최적화

### 파일 형식
- **JPEG** (권장) - 사진/스캔 이미지
- **PNG** (선택) - 투명 배경이 필요하거나 텍스트가 많은 경우

### 파일 크기
- JPEG 품질: 85-90%
- 목표 파일 크기: 150-400KB
- 최대: 600KB 이하

### 스캔 권장사항
- DPI: 200-300 (출력용 아님, 웹용)
- 컬러 모드: RGB (CMYK 아님)
- 배경: 가능하면 깨끗한 흰색 배경

### 최적화 도구
- TinyPNG (https://tinypng.com/)
- Squoosh (https://squoosh.app/)
- ImageOptim (Mac)
- Adobe Photoshop (내보내기 시 품질 조절)

## 📸 촬영/스캔 팁

### 촬영 방법
1. **조명**: 균일한 자연광 또는 링 라이트
2. **각도**: 정면에서 평행하게 촬영
3. **배경**: 단색 (흰색 또는 밝은 회색)
4. **초점**: 상장의 중앙에 맞춤
5. **반사 방지**: 유리액자는 제거 후 촬영

### 스캔 방법
1. **스캐너 사용** (가장 권장)
2. **해상도**: 200-300 DPI
3. **컬러**: 24bit RGB
4. **파일 형식**: JPEG 또는 PNG

### 후보정
- **밝기/대비**: 적당히 조절 (과하지 않게)
- **크롭**: 여백 최소화, 상장만 남기기
- **회전**: 정확히 수평 맞추기
- **색온도**: 자연스러운 색감 유지

## 🎨 카드 디자인 특징

현재 적용된 수상 카드 디자인:

### 레이아웃
- **2단 구성**: 왼쪽 상장 이미지 + 오른쪽 정보
- **골드 테마**: 노란색/금색 그라데이션 배경
- **반응형**: 모바일에서는 1단 레이아웃

### 시각 효과
- **호버 애니메이션**: 
  - 카드 약간 상승
  - 이미지 확대 및 회전
  - 그림자 강조
- **골드 프레임**: 금색 테두리로 수상 강조
- **펄스 효과**: 아이콘이 천천히 맥동

### 색상
- 배경: 크림색/골드 그라데이션
- 텍스트: 진한 갈색/골드톤
- 테두리: 골드 (#ffd700)

## 📞 문제 해결

### 이미지가 안 보일 때
1. 파일 경로 확인
2. 파일명 대소문자 확인
3. 브라우저 캐시 삭제
4. 콘솔 에러 확인

### 이미지가 흐릿할 때
- 해상도를 800px 이상으로 증가
- JPEG 품질을 90-95%로 상향
- 원본을 재스캔

### 로딩이 느릴 때
- 파일 크기를 400KB 이하로 압축
- WebP 포맷 고려
- Lazy loading 확인 (이미 적용됨)
